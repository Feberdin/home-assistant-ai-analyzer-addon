# Home Assistant AI Analyzer

Analyze your Home Assistant installation and generate structured findings for:

- automations and scripts
- entities and references
- templates and performance risks
- integrations and runtime observations
- person geolocation history and movement patterns
- improvement suggestions and optional AI proposals

The add-on provides a small ingress dashboard, a local people map view with OpenStreetMap tiles, a built-in chat assistant for findings and YAML ideas, and machine-readable reports in `/data/analysis`.
